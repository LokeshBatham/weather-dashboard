# Weather Dashboard

This project is a weather dashboard application that allows users to view weather information for various locations.

## Features
- Displays current weather conditions.
- Search functionality for cities around the world.
- User-friendly interface with real-time updates.

## Getting Started

### Prerequisites
- **Node.js**: Make sure you have [Node.js](https://nodejs.org/) installed (version 12.x or higher).
- **npm**: The project uses npm to manage dependencies (npm is included with Node.js).

### Installation

1. Clone the repository:
   git clone <repository-url>

2. Navigate to the project directory:
    cd weather-dashboard

3. Install the project dependencies:
    npm i 
    or 
    npm i --force

4. Set up environment variables:
    REACT_APP_WEATHER_API_KEY=your-api-key

5. Running the Application:
    npm start
